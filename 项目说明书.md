# VibeCoding 项目说明书

## 项目名称及简介

**项目名称：** 梦境宇宙 - AI梦境记录与解析器

**项目简介：**
梦境宇宙是一款基于Web技术的**AI驱动梦境探索应用**，突破传统日记类应用的功能边界，将心理学、数据可视化与人工智能相结合。用户可通过语音实时记录梦境，AI自动解析梦境象征意义、分析情绪倾向，并生成个性化创意故事。应用创新性地引入**梦境模式识别算法**和**集体梦境宇宙**概念，让个人潜意识探索与集体智慧连接成为可能。

**核心创新点：**
- 🎯 **本地AI解析数据库**：内置20+梦境象征符号库，实现离线智能解析
- 🎤 **Web Speech API语音交互**：边说边转文字，零门槛记录梦境
- 🔮 **梦境模式识别**：自动发现情绪规律、重复梦境、预言性梦境
- 🌌 **集体梦境宇宙**：多用户梦境AI串联生成连贯故事
- 📊 **D3.js数据可视化**：力导向图展示梦境元素关联网络

## 技术栈

- **前端框架：** 原生 HTML5 + CSS3 + JavaScript (ES6+)
- **数据可视化：** D3.js v7
- **语音识别：** Web Speech API
- **数据存储：** LocalStorage
- **UI设计：** 深色梦幻风格，移动端优先响应式设计

## 浏览器兼容性

- ✅ Chrome 90+ （推荐，语音功能完整支持）
- ✅ Edge 90+
- ✅ Safari 14.5+ （iOS，语音功能支持）
- ⚠️ Firefox （语音识别功能不支持）
- ❌ QQ浏览器/百度浏览器/UC浏览器 （Web Speech API不支持）

## 项目亮点

1. **全前端实现**：无需后端服务器，纯浏览器端运行，数据本地存储
2. **本地AI驱动**：内置梦境解析数据库，无需联网即可智能解析
3. **语音交互创新**：Web Speech API实现实时语音转文字，边说边录
4. **数据可视化**：D3.js力导向图展示梦境元素关联网络
5. **梦境模式挖掘**：自动分析情绪规律、重复梦境、预言性梦境
6. **社交功能**：集体梦境共享与AI串联故事生成
7. **收藏的解析**：支持保存和查看历史AI解析结果
